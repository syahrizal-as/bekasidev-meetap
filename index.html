Fitur Unggulan Proyek

Apa Yang Akan Anda Bangun?

Anda tidak hanya belajar teori, tapi juga membangun aplikasi utuh yang mengintegrasikan fitur asisten cerdas berbasis AI-Native:

PDF Parser & Chunking

Ekstraksi teks file PDF draf skripsi secara asinkron lalu dipecah menjadi potongan paragraf terstruktur.

Logic & Reference Check

Analisis alur logika antar bab skripsi secara kritis serta validasi kecocokan sitasi teks dengan Daftar Pustaka.

Simulasi Pertanyaan Sidang

AI memprediksi pertanyaan kritis yang potensial ditanyakan oleh dosen penguji sidang untuk melatih kesiapan mental.

WebSockets Real-time

Pemberitahuan instan tanpa refresh halaman saat pemrosesan analisis AI di background selesai menggunakan Laravel Reverb.

Roadmap Belajar

Silabus Lengkap & Terarah

Dari inisialisasi framework hingga integrasi kecerdasan buatan kelas industri.

Part 1

Pendahuluan & Persiapan Proyek

Instalasi Laravel 13, perkenalan fitur Native AI SDK, serta setup library PDF Parser.

Part 2

Arsitektur Database & Model

Perancangan ERD untuk theses, sections, dan AI feedback menggunakan kolom Vector Cast `AsVector`.

Part 3

UI & Laravel Breeze

Setup Auth Breeze dengan Vue 3, Tailwind 4, dan kustomisasi Layout Sidebar menggunakan shadcn-vue.

Part 4

PDF Parsing & Background Job

Logika ekstraksi teks PDF (`PdfService`) dan pembagian teks ke antrean secara background via Queue Job.

Part 5

Membangun Mentor AI

Membuat AI Agent (`ThesisChatAgent`), penanganan sesi diskusi terautentikasi, dan perenderan Markdown di Vue.

Part 6

Queue Worker & Background AI

Memproses pemicu analisis AI secara asinkron agar server tidak freeze menggunakan antrean database.

Part 7

Logic Link & Reference Checker

Melatih AI untuk memeriksa benang merah antar bab serta mencocokkan sitasi draf skripsi dengan bibliografi.

Part 8

Simulasi Pertanyaan Sidang

Memanfaatkan LLM untuk memprediksi 3-5 pertanyaan tersulit saat sidang berdasarkan draf yang diunggah.

Part 9

Dashboard Analisis Vue

Menampilkan metrik Academic Score, ringkasan AI, dan daftar ulasan revisi terstruktur secara interaktif.

Part 10

Interactive Chat with Thesis

Implementasi ruang diskusi real-time di mana mahasiswa dapat mengajukan pertanyaan langsung terkait skripsi mereka.

Part 11

Optimasi RAG & Vector Search

Memanfaatkan Eloquent Vector Search (`nearestTo`) untuk menghemat token API dan meningkatkan akurasi jawaban obrolan.

Part 12

Manajemen User & Global Toast

Membuat sistem CRUD untuk administrator pengguna serta integrasi pesan notifikasi melayang (Toast) via Inertia flash.

Part 13

Real-time WebSockets dengan Reverb

Integrasi Laravel Reverb untuk mendorong pembaruan status pemrosesan skripsi dari server ke browser secara instan.
